정수론
- 정수의 성질을 연구하는 분야
- 약수와 배수

약수
- 특정 정수를 나누어 떨어지게 하는 수
ex) 20 => 1 2 4 5 10 20

소수
- 약수가 1과 자기자신뿐인 정수
ex) 7 => 1  2  3  4  5  6  7
            2 -------- n-1  => 이 숫자들에 대해서 하나도 나누어 떨어지지 않는 지 확인

공약수와 공배수
* A와 B의 공약수
- A의 약수이면서 B의 약수
* A와 B의 공배수
- A의 배수이면서 B의 배수

* 최대공약수 (GCD : Greatest Common Divisior)
- 공약수중 최댓값
* 최소공배수 (LCM : Lowest Common Multiplier)
- 공배수충 최솟값

* 유클리드 호제법
- 최대 공약수를 구하기 위한 알고리즘

(A > B 일 때) A % B = r(A를 B로 나눈 나머지)
gcd(A, B) = gcd(B, r)

lcm(A, B) = (A / GCD) * (B / GCD) * GCD
lcm(A, B) = A * B / GCD
